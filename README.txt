#O que o projeto faz?

Projeto criado para fazer o pré-processamento de bulas de medicmanetos da base da ANVISA em português, utilizando Expressões regualres e a Biblioteca Spacy

#Como ele funciona?

As bulas em .PDF devem ser compactadas e colocadas na pasta "entrada" do projeto, na pasta saida_segmentada estará os arquivos compactados JSON segmentados.

#Depedencias a serem instaladas

python -m spacy download pt_core_news_lg

#Rodar o projeto

python main.py
